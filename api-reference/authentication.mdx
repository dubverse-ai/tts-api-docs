## API Key Authentication

Dubverse AI uses API key authentication to secure and identify your API requests. To authenticate your requests, you need to include your API key in the headers of each API call.

### Obtaining Your API Key

1. Log in to your Dubverse AI account
2. Navigate to the API section in your dashboard
3. Generate or copy your API key

<Warning>
  Keep your API key secure and do not share it publicly. If you believe your key has been compromised, regenerate it immediately from your dashboard.
</Warning>
